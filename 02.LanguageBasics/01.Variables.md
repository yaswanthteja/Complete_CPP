



## What is an Identifier?

An identifier is the **name** you give to elements in your program such as variables, functions, classes, arrays, etc.

* **Purpose** : It uniquely identifies these entities so you can reference them in your code.
* **Examples** : `age`, `salary`, `main`, `Student`, `calculateSum`.

# Variables

 In C++, a **variable** is a named storage location in memory that holds a value. It has a **type** (like `int`, `double`, `char`, `string`, `bool`) which defines what kind of data it can store, and a **name** (identifier) that you use to access it.

## Technical Definition

* A **variable** is a container for storing data values.
* **Syntax** :

```
  type variableName = value;
```

* `type` â†’ data type (e.g., `int`, `double`)
* `variableName` â†’ identifier (e.g., `x`, `age`)
* `value` â†’ optional initial value

## ğŸ”¹ Types of Variables in C++

Common built-in types include:

| Type       | Description                         | Example                       |
| ---------- | ----------------------------------- | ----------------------------- |
| `int`    | Stores integers (whole numbers)     | `int age = 25;`             |
| `double` | Stores floating-point numbers       | `double pi = 3.14;`         |
| `char`   | Stores single characters            | `char grade = 'A';`         |
| `string` | Stores text (requires `<string>`) | `string name = "Yaswanth";` |
| `bool`   | Stores true/false values            | `bool isReady = true;`      |

## ğŸ”¹ Variable Declaration vs Initialization

* **Declaration** : Telling the compiler about the variable.
  cpp

```
  intnum;   // declared but not initialized
```

* **Initialization** : Assigning a value at the time of declaration.
  cpp

```
  intnum = 10;   // declared and initialized
```

You can also assign later:

cpp

```
num = 20;   // value updated
```

## ğŸ”¹ Key Notes

* **Scope** : Variables exist only within the block `{}` they are defined in.
* **Lifetime** : Depends on where they are declared (local vs global).
* **Naming rules** : Must start with a letter or underscore, cannot use reserved keywords.
* **Multiple variables** : You can declare multiple of the same type in one line:
  cpp





## What is I/O in C++?

* **Input** : Taking data from the user (keyboard, file, etc.).
* **Output** : Displaying data to the user (screen, file, etc.).
* In C++, I/O is handled using the **iostream library** (`cin`, `cout`, `cerr`, `clog`).

## ðŸ”¹ Standard Input (`cin`)

* `cin` stands for  **console input** .
* Used to read data from the keyboard.
* Syntax:

  ```cpp
  cin >> variable;
  ```
* Example:

  ```cpp
  int age;
  cout << "Enter your age: ";
  cin >> age;
  cout << "You entered: "<< age << endl;
  ```

## ðŸ”¹ Standard Output (`cout`)

* `cout` stands for  **console output** .
* Used to print data to the screen.
* Syntax:

  ```cpp
  cout << "Message";
  ```
* Example:

  ```cpp
  cout << "Hello, Yaswanth!"<< endl;
  ```

## ðŸ”¹ Other Output Streams

* `cerr` â†’ Prints error messages (unbuffered, appears immediately).

  ```cpp
  cerr << "Error: Invalid input!"<< endl;
  ```
* `clog` â†’ Prints log messages (buffered, may delay until flushed).


  ```cpp
  clog << "Program started successfully."<< endl;
  ```

## ðŸ”¹ Example Program


```cpp
#include <iostream>
using namespacestd;

int main() {
    string name;
    int age;

    cout << "Enter your name: ";
    cin >> name;   // input

    cout << "Enter your age: ";
    cin >> age;    // input

    cout << "Hello "<< name << ", you are "<< age << " years old."<< endl; // output

    return 0;
}
```

**Output (example run):**

Code

```
Enter your name: Yaswanth
Enter your age: 22
Hello Yaswanth, you are 22 years old.
```

## âœ… Key Takeaways

* `cin` â†’ input from keyboard.
* `cout` â†’ output to screen.
* `cerr` â†’ error messages.
* `clog` â†’ log/debug messages.
* All are part of the `<iostream>` library.

# Advanced



## File I/O in C++

C++ provides the `<fstream>` library for working with files. It has three main classes:

* `ifstream` â†’ Input file stream (read from files).
* `ofstream` â†’ Output file stream (write to files).
* `fstream` â†’ File stream (read and write both).

## ðŸ”¹ Writing to a File (`ofstream`)


```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream outFile("data.txt");  // open file for writing

    if(!outFile) {
        cerr << "Error opening file!"<< endl;
        return 1;
    }

    outFile << "Hello, Yaswanth!"<< endl;
    outFile << "Age: 22"<< endl;

    outFile.close();  // always close the file
    return 0;
}
```

ðŸ‘‰ This creates a file `data.txt` and writes text into it.

## ðŸ”¹ Reading from a File (`ifstream`)


```cpp
#include <iostream>
#include <fstream>
#include <string>
using namespace std;

int main() {
    ifstream inFile("data.txt");  // open file for reading
    string line;

    if(!inFile) {
        cerr << "Error opening file!"<< endl;
        return 1;
    }

    while(getline(inFile, line)) {
        cout << line << endl;  // print each line
    }

    inFile.close();
    return 0;
}
```

ðŸ‘‰ This reads the contents of `data.txt` line by line and prints them.

## ðŸ”¹ Reading & Writing (`fstream`)


```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    fstream file("data.txt", ios::in | ios::out | ios::app);

    if(!file) {
        cerr << "Error opening file!"<< endl;
        return 1;
    }

    file << "New line added!"<< endl;  // write
    file.seekg(0);                      // move to beginning
    string line;
    while(getline(file, line)) {
        cout << line << endl;           // read
    }

    file.close();
    return 0;
}
```

## ðŸ”¹ File Modes

When opening files, you can specify modes:

* `ios::in` â†’ read
* `ios::out` â†’ write
* `ios::app` â†’ append
* `ios::trunc` â†’ overwrite existing file
* `ios::binary` â†’ binary mode

## âœ… Key Takeaways

* Use `<fstream>` for file I/O.
* `ifstream` â†’ read, `ofstream` â†’ write, `fstream` â†’ both.
* Always check if the file opened successfully (`if (!file)` check).
* Always close files after use (`file.close()`).
